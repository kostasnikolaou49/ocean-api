## Installation (Development)

```bash
$ npm install
```

## Migration database

```bash
# migration tables
$ npm run schema:sync

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Environment
### .env
```bash

PORT=4000
```

## API documentation
To see the API documentation, please checkout `localhost:PORT/api`.

## How to Setup
1. create and set environemnt in .env
2. run npm install
3. run npm run start:dev
